# OpenAI — State of Enterprise AI (2025)

## Summary
OpenAI's enterprise report documenting usage growth and productivity effects across ChatGPT Enterprise customers through 2025. Reports 8x growth in ChatGPT message volume since November 2024 and 320x growth in API reasoning-token consumption. Active ChatGPT Enterprise users save 40–60 minutes per day; 75% of workers report improved speed or quality. The key distributional finding: frontier organizations send 6x more messages than the median enterprise — a widening capability gap inside the same tooling.

## Why it matters for the CMM
Primary evidence for the widening Level 4 vs. Level 1 gap within a single vendor's customer base. The 6x frontier-vs-median ratio is the cleanest empirical anchor for "the same tool produces vastly different org-level outputs depending on maturity." The report also supplies the individual-productivity figures that organizations at any level are likely to encounter and need to contextualize — individual time savings do not aggregate into enterprise value without the workflow layer that Levels 3+ provide.

## Claims citable from this observation
- ChatGPT message volume grew 8x from November 2024
- API reasoning-token consumption grew 320x
- 40–60 minutes per day saved per active ChatGPT Enterprise user
- 75% of workers report improved speed or quality
- Frontier organizations send 6x more messages than median enterprises
- "75% of workers report being able to complete tasks they previously could not perform, including programming support and code review, spreadsheet analysis and automation, technical tool development and troubleshooting, and custom GPT or agent design." (verbatim, OpenAI)
